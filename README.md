# mul_tab
prints multiplication tables

<h2>Binaries</h2>
In the <i>bin</i> folder, only for Linux x86_64.

<h2>Sources</h2>
In the <i>sources</i> folder.

<h2>How to use it ?</h2>
Here's the syntax : <pre>./mul_tab [i]</pre>
With <i>i</i> an integer between 0 and 9.
